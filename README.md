## Running Locally

Clone the project

```bash
git clone https://github.com/VisionarySuperstar/CleanPortfolio.git
```

Go to the project directory

```bash
cd CleanPortfolio.git
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
